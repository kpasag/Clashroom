Lecture Bingo – Game Rules

1. Overview

Lecture Bingo is a multiplayer game where each player receives a unique bingo card filled with common lecture scenarios. As the lecture happens in real time, players mark off squares when the corresponding scenario occurs. Get a full row or column before your opponent to win!

2. Getting Started

Join or create a game. Once both players are connected, each is given a randomly generated bingo card. Do not refresh or leave the page during a game, or your card may be lost.

3. How to Play

Watch the lecture. Each square describes a scenario (e.g., “Professor goes off-topic,” “Someone asks to repeat the slide,” “Technical issue occurs,” etc.). Mark a square by tapping/clicking it only when the scenario has actually happened. Once marked, squares cannot be unmarked—so be honest!

4. Winning the Game

  • You complete any full row
  • Any full column

When a player completes one, the game immediately ends and the winner is declared.

5. Fair Play & Etiquette

  • Only mark squares for events that genuinely occur during the lecture.
  • Don’t reveal your card to the other player unless you want to!
  • If a disputed event happens, the game relies on player honesty—Lecture Bingo is meant to be fun, not competitive.

6. End of Game

  • Start a new round with fresh random cards
  • Return to the main menu
